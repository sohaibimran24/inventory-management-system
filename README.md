# 📦 Inventory Management System

A Python-based Inventory Management System built using Object-Oriented Programming (OOP) principles.

## 🚀 About The Project
This system allows businesses to efficiently manage their inventory by tracking products, stock levels, and updates — all through a clean Python-based interface.

## ✨ Features
- Add, update, and delete products
- Track stock levels in real time
- Search and filter inventory items
- Built with clean OOP architecture (classes, inheritance, encapsulation)

## 🛠️ Built With
- Python 3
- Object-Oriented Programming (OOP)
